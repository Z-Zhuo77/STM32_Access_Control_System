# STM32_Access_Control_System

基于STM32F103C8T6的门禁控制系统，实现指纹识别、语音播报、OLED显示等功能。

## 项目介绍

这是一个基于STM32F103C8T6单片机的门禁控制系统项目，包含以下功能模块：

- ✅ **AS608指纹模块**：支持指纹录入、验证、删除，支持等待手指放置后自动检测
- ✅ **SYN6288语音模块**：支持中文语音播报，操作提示清晰
- ✅ **OLED显示模块**：128x64屏幕，实时显示操作状态和提示信息
- ✅ **矩阵键盘**：16键键盘输入，支持各种功能操作
- ✅ **HX711重量传感器**：压力检测功能测试完成

## 硬件连接详细说明

### AS608指纹模块连接
- **VCC** → STM32 3.3V
- **GND** → STM32 GND
- **TX** → STM32 PA10 (USART1_RX)
- **RX** → STM32 PA9 (USART1_TX)
- **GND** → STM32 GND（第二个GND）
- **INT** → 可选连接GPIO用于检测手指放置

### SYN6288语音模块连接
- **VCC** → STM32 5V（注意：部分模块需要5V供电）
- **GND** → STM32 GND
- **TX** → STM32 PB11 (USART3_RX)
- **RX** → STM32 PB10 (USART3_TX)

### OLED显示模块连接
- **VCC** → STM32 3.3V
- **GND** → STM32 GND
- **SCL** → STM32 PB6 (I2C1_SCL)
- **SDA** → STM32 PB7 (I2C1_SDA)

### 矩阵键盘连接
- **行线**：PA0, PA1, PA4, PA5
- **列线**：PA6, PA7, PB0, PB1

### HX711重量传感器连接
- **VCC** → STM32 3.3V
- **GND** → STM32 GND
- **DT** → STM32 PB12
- **SCK** → STM32 PB13

### 蜂鸣器连接
- **VCC** → STM32 3.3V
- **GND** → 通过三极管连接到STM32 PB14

## 功能特点

- ✅ 语音播报清晰，操作有语音提示

## 操作说明

### 按键功能

- **KEY_1**：蜂鸣器测试
- **KEY_2**：重量传感器测试
- **KEY_3**：原始数据显示
- **KEY_4**：清除屏幕
- **KEY_5**：指纹注册（等待手指放置）
- **KEY_6**：指纹验证（等待手指放置）
- **KEY_7**：清除指纹库
- **KEY_8**：预留功能
- **KEY_9**：预留功能
- **KEY_0**：预留功能
- **KEY_A**：预留功能
- **KEY_B**：预留功能
- **KEY_C**：预留功能
- **KEY_D**：预留功能
- **KEY_STAR**：预留功能
- **KEY_POUND**：返回主菜单

### 指纹操作流程

1. **指纹注册**：
   - 按KEY_5，系统提示"请放手指"
   - 放置手指后自动采集第一次图像
   - 提示"请移开手指"
   - 再次提示"请放手指"
   - 放置手指后自动采集第二次图像
   - 完成注册并显示ID

2. **指纹验证**：
   - 按KEY_6，系统提示"请放手指"
   - 放置手指后自动验证
   - 显示验证结果和ID
